# APIs

Welcome to the Dotshell APIs documentation. Here you will find comprehensive documentation for all our public APIs.

## Available APIs

- [Pelo API](pelo-api) - A simple API mirror for real-time data of the Lyon TCL (public transport) network.

## Getting Started

To use any of our APIs, you will need an API key. Please contact us to obtain your API key.

## Support

If you encounter any issues or have questions about our APIs, please open an issue on our [GitHub repository](https://github.com/dotshell-org/documentation).
