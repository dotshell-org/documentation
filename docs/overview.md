Welcome to the **Dotshell Documentation** portal. Here you will find comprehensive guides and technical specifications for all our open-source mobility and transit libraries.

Dotshell provides advanced libraries and pipelines to simplify public transit integration, route optimization, and GTFS data processing.

## Getting Started

Explore our core components in the sidebar:
- **[raptor-gtfs-pipeline](/libraries/raptor/raptor-gtfs-pipeline/)**: A powerful toolchain for processing GTFS data, validating service periods, and compiling binary transit feeds.
- **[raptor-kt](/libraries/raptor/raptor-kt/)**: A high-performance Kotlin implementation of the Round-Based Public Transit Routing (RAPTOR) algorithm, optimized for android and server environments.
